# 💪 Landing Page - Personal Trainer

Uma Landing Page moderna, responsiva e otimizada para conversão, desenvolvida para profissionais de Personal Trainer que desejam fortalecer sua presença digital, captar novos alunos e apresentar seus serviços de forma profissional.

O projeto foi construído utilizando apenas **HTML5**, **CSS3** e **JavaScript**, sem frameworks, garantindo alta performance, código limpo e fácil manutenção.
